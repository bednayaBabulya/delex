# Тестовое задание — Веб-сайт Delex
 
## Описание проекта

Небольшой одностраничный сайт, выполненный в рамках тестового задания. Проект демонстрирует знание базовых технологий фронтенда — **HTML**, **CSS** и **JavaScript**.

## Функциональность

* Семантическая разметка HTML5
* Адаптивная вёрстка с использованием Flexbox / Grid
* Минимальная интерактивность на чистом JavaScript (валидация, анимации, переключатели и т.п.)
* Кроссбраузерная совместимость

## Структура проекта

```
/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── script.js
└── README.md
```

## Запуск проекта

1. Склонируйте репозиторий:

   ```bash
   git clone https://github.com/<username>/<repo>.git
   ```
2. Откройте файл `index.html` в браузере.
   (Никаких дополнительных зависимостей не требуется.)
